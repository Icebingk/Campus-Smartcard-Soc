# 校园智能卡 SoC — 统一外设地址映射表 (Memory Map)

> **版本**: v1.0  
> **日期**: 2026-06-14  
> **负责人**: 阿呆不呆  
> **CPU**: RV32EC (32-bit, 4GB 寻址空间)

## 1. 总体地址分区

```
┌──────────────────────────────────────────────────────┐
│ 0xFFFF_FFFF                                           │
│                    保留 (Reserved)                     │
│                                                       │
│ 0x4000_4000 ──────────────────────────────────────── │
│               APB 外设区 (4KB x N Slaves)              │
│ 0x4000_0000 ──────────────────────────────────────── │
│                    保留                                │
│ 0x0001_2000 ──────────────────────────────────────── │
│              SRAM (8KB)  数据/堆栈                      │
│ 0x0001_0000 ──────────────────────────────────────── │
│                    保留                                │
│ 0x0000_4000 ──────────────────────────────────────── │
│              ROM  (16KB) 固件/启动代码                   │
│ 0x0000_0000 ──────────────────────────────────────── │
└──────────────────────────────────────────────────────┘
```

## 2. 详细地址分配

### 2.1 ROM（启动存储器）

| 属性 | 值 |
|------|-----|
| **基址** | `0x0000_0000` |
| **大小** | 16 KB (0x4000) |
| **地址范围** | `0x0000_0000` ~ `0x0000_3FFF` |
| **总线** | AHB-Lite Slave 0 |
| **说明** | 存放 CPU 启动向量、boot code、固化交易协议栈 |

### 2.2 SRAM（数据存储器）

| 属性 | 值 |
|------|-----|
| **基址** | `0x0001_0000` |
| **大小** | 8 KB (0x2000) |
| **地址范围** | `0x0001_0000` ~ `0x0001_1FFF` |
| **总线** | AHB-Lite Slave 1 |
| **说明** | 数据段、堆栈、防冲突临时缓存、AES 明文/密文缓冲区 |

### 2.3 APB 外设区（通过 AHB2APB Bridge）

> **AHB 基址**: `0x4000_0000`  
> **每外设分配**: 4 KB 地址空间  
> **APB 地址线**: `paddr[15:0]`，其中 `paddr[15:12]` 用于 Slave 选择

| Slave 编号 | 外设 | APB 基址 | 地址范围 | 负责人 |
|-----------|------|----------|----------|--------|
| **0** | 数字基带 (Baseband) | `0x4000_0000` | `0x4000_0000` ~ `0x4000_0FFF` | 林子轩 |
| **1** | AES-128 引擎 | `0x4000_1000` | `0x4000_1000` ~ `0x4000_1FFF` | 梁芷晴 |
| **2** | EEPROM 控制器 | `0x4000_2000` | `0x4000_2000` ~ `0x4000_2FFF` | 阿呆不呆 |
| **3** | 系统控制/PMU | `0x4000_3000` | `0x4000_3000` ~ `0x4000_3FFF` | 阿呆不呆 |
| **4-15** | 预留扩展 | `0x4000_4000` ~ `0x4000_FFFF` | — |

## 3. 各外设寄存器文件定义

### 3.1 数字基带 (Baseband) — 0x4000_0000

| 偏移 | 寄存器名 | 位宽 | 访问 | 描述 |
|------|----------|------|------|------|
| `0x000` | `BB_CTRL` | 32 | RW | 控制寄存器（使能、模式选择） |
| `0x004` | `BB_STATUS` | 32 | RO | 状态寄存器（防冲突完成、错误码） |
| `0x008` | `BB_TX_DATA` | 32 | RW | 发送 FIFO 数据 |
| `0x00C` | `BB_RX_DATA` | 32 | RO | 接收 FIFO 数据 |
| `0x010` | `BB_FIFO_LEVEL` | 32 | RO | FIFO 水位（Tx/Rx 可用字节数） |
| `0x014` | `BB_INT_EN` | 32 | RW | 中断使能 |
| `0x018` | `BB_INT_STATUS` | 32 | RO/W1C | 中断状态（写1清零） |
| `0x01C` | `BB_BAUD_CFG` | 32 | RW | 波特率/定时器配置 |
| `0x020` ~ `0xFFF` | — | — | — | 保留 |

### 3.2 AES-128 引擎 — 0x4000_1000

| 偏移 | 寄存器名 | 位宽 | 访问 | 描述 |
|------|----------|------|------|------|
| `0x000` | `AES_CTRL` | 32 | RW | 控制（加密/解密选择、启动、模式） |
| `0x004` | `AES_STATUS` | 32 | RO | 状态（忙、完成、错误） |
| `0x008` | `AES_KEY0` | 32 | RW | 密钥 [31:0] |
| `0x00C` | `AES_KEY1` | 32 | RW | 密钥 [63:32] |
| `0x010` | `AES_KEY2` | 32 | RW | 密钥 [95:64] |
| `0x014` | `AES_KEY3` | 32 | RW | 密钥 [127:96] |
| `0x018` | `AES_DIN0` | 32 | RW | 明文/密文输入 [31:0] |
| `0x01C` | `AES_DIN1` | 32 | RW | 明文/密文输入 [63:32] |
| `0x020` | `AES_DIN2` | 32 | RW | 明文/密文输入 [95:64] |
| `0x024` | `AES_DIN3` | 32 | RW | 明文/密文输入 [127:96] |
| `0x028` | `AES_DOUT0` | 32 | RO | 结果输出 [31:0] |
| `0x02C` | `AES_DOUT1` | 32 | RO | 结果输出 [63:32] |
| `0x030` | `AES_DOUT2` | 32 | RO | 结果输出 [95:64] |
| `0x034` | `AES_DOUT3` | 32 | RO | 结果输出 [127:96] |
| `0x038` ~ `0xFFF` | — | — | — | 保留 |

### 3.3 EEPROM 控制器 — 0x4000_2000

| 偏移 | 寄存器名 | 位宽 | 访问 | 描述 |
|------|----------|------|------|------|
| `0x000` | `EEP_CTRL` | 32 | RW | 控制（读写选择、启动） |
| `0x004` | `EEP_STATUS` | 32 | RO | 状态（忙、完成、错误） |
| `0x008` | `EEP_ADDR` | 32 | RW | EEPROM 字节地址 |
| `0x00C` | `EEP_WDATA` | 32 | RW | 写数据 |
| `0x010` | `EEP_RDATA` | 32 | RO | 读数据 |
| `0x014` | `EEP_LEN` | 32 | RW | 传输字节数 |
| `0x018` ~ `0xFFF` | — | — | — | 保留 |

### 3.4 系统控制/PMU — 0x4000_3000

| 偏移 | 寄存器名 | 位宽 | 访问 | 描述 |
|------|----------|------|------|------|
| `0x000` | `SYS_CTRL` | 32 | RW | 系统控制（时钟门控使能、外设复位） |
| `0x004` | `SYS_STATUS` | 32 | RO | 系统状态 |
| `0x008` | `SYS_CLK_DIV` | 32 | RW | 时钟分频配置 |
| `0x00C` | `SYS_CHIP_ID` | 32 | RO | 芯片 ID（版本号） |
| `0x010` ~ `0xFFF` | — | — | — | 保留 |

## 4. AHB-Lite 地址译码方案

| HSEL | 目标 | 地址匹配条件 |
|------|------|-------------|
| `HSEL_ROM` | ROM | `HADDR[31:14] == 18'h0` (低 16KB) |
| `HSEL_SRAM` | SRAM | `HADDR[31:13] == 19'h0000_8000 >> 13` |
| `HSEL_APB` | AHB2APB Bridge | `HADDR[31:16] == 16'h4000` |
| (默认) | — | 返回 `HRESP=ERROR` |

## 5. CPU 启动约定

| 项目 | 值 |
|------|-----|
| **复位向量** | `0x0000_0000` (ROM 基址) |
| **中断向量基址** | `0x0000_0010` (ROM 内偏移 16 字节) |
| **栈指针初始化** | `0x0001_1FFC` (SRAM 顶部，8KB-4) |
